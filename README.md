# 智慧实验室设备预约与管理系统

## 项目简介
本系统基于 C++ / Qt / MySQL 开发，实现实验室设备的在线预约、智能冲突检测、状态监控、数据统计与可视化功能，解决传统实验室管理效率低、预约混乱、设备状态不透明等问题。

## 技术栈
- C++
- Qt 5 / Qt 6
- MySQL 8.0
- QCustomPlot（数据可视化）
- Git + GitHub（团队协作与版本管理）

## 功能模块
- 用户登录与权限管理（学生/管理员/系统管理员）
- 设备信息管理（增删改查、图片上传）
- 设备预约与智能冲突检测
- 设备状态实时监控
- 使用记录与故障上报
- 数据统计与图表可视化
- 公告发布与消息提醒

## 团队分工
- 成员A：Qt 界面开发、UI交互、图表展示
- 成员B：C++ 后端逻辑、预约调度、业务核心
- 成员C：数据库设计、SQL脚本、测试与文档
