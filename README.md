# 🧪 智慧实验室设备预约与管理系统

> 基于 C++ / Qt / MySQL 开发的实验室设备预约与管理平台，提供设备在线预约、智能冲突检测、状态监控、数据统计与可视化功能，解决传统实验室管理效率低、预约混乱、设备状态不透明等问题。

## 📌 项目简介

本系统面向高校及科研实验室，实现设备资源的数字化管理。学生/教师可在线预约设备、查看使用记录、上报故障；管理员可审核预约、管理设备、发布公告；系统管理员可进行用户权限管理及数据统计。系统采用 Qt 构建跨平台桌面客户端，MySQL 作为后端数据库，并通过智能冲突检测算法避免预约超订。

## ✨ 功能特性

- 🔐 **多角色权限控制**  
  支持学生、设备管理员、系统管理员三种角色，不同角色拥有不同的操作界面与权限。
- 🖥️ **设备信息管理**  
  设备的增删改查、图片上传、状态标记（空闲/使用中/维修中）。
- 📅 **设备预约与智能冲突检测**  
  用户可按时间段查询设备空闲状态，系统自动检测时间重叠并阻止冲突预约。
- 📡 **设备状态实时监控**  
  管理员可查看每台设备的当前状态、使用人及使用时段。
- 📝 **使用记录与故障上报**  
  用户可查看个人预约历史，提交故障报修；管理员处理报修并记录闭环。
- 📊 **数据统计与图表可视化**  
  基于 QCustomPlot 展示设备使用率、用户预约频次、故障统计等图表。
- 📢 **公告发布与消息提醒**  
  管理员发布实验室通知，系统对预约成功/驳回/故障处理进行消息提醒。

## 🧰 技术栈

| 层级         | 技术                                      |

| 客户端界面   | C++ + Qt 5 / Qt 6 (Widgets)               |

## ⚙️ 环境要求

- **操作系统**：Windows 10/11 / Linux (Ubuntu 20.04+) / macOS 10.15+
- **编译器**：MinGW 7.3+ / MSVC 2019+ / GCC 7+（需支持 C++11 或更高）
- **Qt 版本**：Qt 5.12 ～ 6.x（安装时需勾选 `Qt Sql` 模块）
- **MySQL**：8.0 及以上（或 MariaDB 10.5+）
- **Qt MySQL 驱动**：  
  Windows 用户需将 MySQL 安装目录下的 `libmysql.dll` 复制到 Qt 编译器的 `bin` 目录中（例如 `C:\Qt\5.15.2\mingw81_64\bin`）。
